# ACID Microbiome
### Gut enterotype- and body mass index (BMI)-dependent effects of anthocyanin supplementation on gut microbiota composition in individuals at risk for cognitive decline: A randomized placebo-controlled trial
## Scripts
* ***alpha_diversity.R:*** Calculates and statistically tests various alpha diversity metrics.
* ***alphadiversity_plot.R:*** Generates plots for different alpha diversity metrics.
* ***baseline_enterotype_response.R:*** Analyzes the effect of baseline enterotype on intervention response.
* ***beta_diversity.R:*** Computes beta diversity measures.
* ***bmi_effect_intervention.R:*** Examines the impact of baseline BMI classification on intervention response.
* ***effectofintervention.R:*** Assesses the intervention's effect on microbiome composition.
* ***enterotype_dmm.R:*** Classifies microbiome samples into distinct enterotypes using Dirichlet Multinomial Mixture (DMM) models.
* ***pre_processing.R:*** Performs data cleaning and preparation.
* ***precustestwoenterotype.R:*** Conducts Procrustes analysis to compare pre- and post-intervention microbial community structures, stratified by enterotype.
* ***top10_taxa.R:*** Generates plots of the top 10 taxa at genus and phylum levels.
